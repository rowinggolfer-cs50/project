CS50 Final Project

This website introduces the final project of Neil Wallace for the Harvard University CS50 introduction to computing course 2020.

Project Aim

I am a dentist, who was prevented from contact with patients during the strict United Kingdom's covid-19 lockdown of April-July 2020. During this period, my colleagues and I shared responsibility for triaging telephone conversations from patients with problems. We needed a way to record and share this information. My final project, therefore, documents the website I created to provide a secure online database frontend.
For obvious reasons I am unable to share the intranet version of the site which contains sensitive information and patient details etc... However I can confirm it worked very well and my colleagues used it to the great benefit of and patients.

Tools used.

The website uses the Python programming language, and the Django Web Framwork.
The database used in production was PostgreSQL
NB. for this demonstration version of the website I am using the lightweight sqllite3 database which was introduced during the problem set 7 of the CS50 course.
